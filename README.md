# Embedded-System 

## Day 1

임베디드 시스템이란?
 - 내장형 시스템
 - 특정기능
 - 제한적인 기능
 - 엄격한 제약사항(돈)
 - 실시간성을 요구
 - 신뢰성
 - 최적화 필요
 - 개발 환경의 이질성

OS란
 - 하드웨어를 컨트롤, 관리하는 소프트웨어

Kernel
 - 너무 중요해서 항상 수행해야 함

master : 통신을 주는 시스템
slave : 통신을 받는 시스템(메모리는 보통 slave임)

interrupt : 뭔가 오면 알림을 주면 그때수행
